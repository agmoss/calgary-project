About

A free and open source tool for rental property analysis!

The Calgary Project consists of two independent software repositories, one for data collection and one for the analytical web application.

The data collection repository is an ETL pipeline written in Python with a MySQL backend. The web application is built with the Django framework and employs JavaScript for graphics and interactivity. Both repositories are dockerized and run in Microsoft Azure. The project encompasses the principles of CI/CD.

Data Collection

Rentfaster.ca houses individual rental listings for major cities in Canada. These listings are transactional records containing features such as price, location, availability, date, square feet etc. The user of Rentfaster.ca has the option to filter on these features to narrow their search to a set of preferences. A specific listing can then be rented by the user right on the webpage.

While this is a great service for prospective renters, there is little ability to gain insight on the rental market as a whole. Calgary lacks a free and accessible service for analyzing the health and status of the rental market. The Calgary Project aims to fulfill this absence by democratizing an understanding of the rental market for all. Rental listing data is collected daily and examined for duplicate entries, inconsistencies, and formatting errors. Additionally, previously collected listings are cross referenced with new listings and archived to create a timeline of historical data.

All listings go through a rigorous engineering process before being stored persistently in a format amenable to analytical queries. The means of collection and storage allow for advanced analysis including time series, supervised and unsupervised learning, spatial analysis, clustering, and forecasting.

Interactive Web Application

Data for the web application is populated from the data collection repository. In its current state, the web app is a proof of concept and serves as a starting point for answering some high level questions about the Calgary rental market. Exploratory visuals offer insight into the rental market by community, property type, amenities offered, and historical time period. The interactivity of the visuals allow for custom ad hoc queries on the latest data. New insights and changes to the market can be observed and analyzed as they happen.

The web app is under active development and is currently in beta stage. New analytical features that incorporate machine learning and forecasting are currently being developed. If you are using this service and you have an idea for a feature, please leave a comment on the contact page or submit a pull request!
