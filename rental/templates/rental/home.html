The Calgary Project

{{ domain.domain }}

A free and open source tool for rental property analysis!



The Calgary Project is a proof of concept application for tracking and analyzing rental listings in Calgary. The foundation of the application is the Rentfaster.ca API. Through a carefull designed ETL process, data from the API is retrieved, cleaned, and stored every day for an up to date and accurate analysis.


Advanced statistical and enginerring procedures transform this data into an immersive visual and analytical experience for the user.
