The Calgary Project

{{ domain.domain }}

A free and open source tool for rental property analysis!



The Calgary Project is a free and open source web app for analyzing Calgary's rental market. The application collects rental listing information from Rentfaster.ca and transforms it into an immersive analytical experience.


Interactive charts, maps, and metrics allow you to evaluate trends, discrepancies, and opportunities in the marketplace. Data is updated daily for a consistently relevant experience.


This project is currently in its infancy. New features and improvements are being added daily. Feedback and user involvement is highly encouraged. Please leave your thoughs, advice, and inquiries in the contact menu.
